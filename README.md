# AI-Powered Email Spam Detection System

## Overview

This project uses Machine Learning and Natural Language Processing (NLP) to classify emails as Spam or Not Spam.

## Features

* Email spam detection using Logistic Regression
* Text preprocessing and feature extraction
* Flask-based web application
* Real-time prediction through a web interface

## Technologies Used

* Python
* Scikit-learn
* Flask
* HTML/CSS
* Git & GitHub

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Text Vectorization using CountVectorizer
4. Model Training using Logistic Regression
5. Model Serialization using Pickle
6. Web Application Development using Flask
7. Deployment using GitHub

## How to Run

1. Install dependencies
2. Run:
   python app.py
3. Open:
   http://127.0.0.1:5000
